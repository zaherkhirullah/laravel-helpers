### INSTALLATION STEPS

<i> Step 1 </i>

```
$ composer require hayrullah/helpers
 ```

<i> Step 2 </i>

```
$ php artisan vendor:publish --provider="Hayrullah\Helpers\HelperServiceProvider" 
$ php artisan vendor:publish --provider="Hayrullah\Helpers\HelperServiceProvider" --tag=helpers
```
